*************************************************************************************************
Partie 2 | Algorithmes de base
*************************************************************************************************

Vue d'ensemble
==============

Cette partie présente les algorithmes **fondamentaux** de la fouille de données
non supervisée : le **clustering** (regrouper des observations similaires), le
**Frequent Itemset Mining** (motifs fréquents + règles d'association) et le
**Sequential Pattern Mining** (motifs fréquents *ordonnés*).

Objectifs
=========

À l'issue de cette partie, chaque étudiant.e sera capable de :

* Expliquer et appliquer le **clustering** (K-means, clustering hiérarchique)
* Formaliser et résoudre le **Frequent Itemset Mining** (support, lattice, a priori)
* Générer des **règles d'association** (confiance, lift) et comprendre les patterns fermés/maximaux
* Étendre la démarche aux **séquences** (*Sequential Pattern Mining*)
