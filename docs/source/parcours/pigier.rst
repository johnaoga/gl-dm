*************************************************************************************************
Parcours PIGIER (M2)
*************************************************************************************************

Ce parcours est la version **fondamentaux** du cours de Data Mining, enseignée à
PIGIER. Il couvre une sélection des chapitres communs, sur **7 séances de 3 h**.

Objectifs
=========

À la fin du cours, l'étudiant.e doit pouvoir :

* Différencier les notions autour du DM (IA, science des données, ML, DL, visualisation, préparation de données)
* Utiliser les outils nécessaires pour s'informer sur un jeu de données et le traiter
* Définir le DM et les concepts associés (*pattern mining*, prédiction, clustering, …)
* Identifier les cas d'usage des algorithmes de DM
* Expliquer le processus **CRISP-DM**
* Interpréter données et graphiques ; faire de la visualisation
* Implémenter des algorithmes (*frequent itemset mining* et clustering)

Chapitres couverts
==================

* :doc:`chap1 — Introduction au data mining <../part1/chap1>`
* :doc:`chap2 — Analyse de données <../part1/chap2>`
* :doc:`chap4 — Outils d'implémentation <../part1/chap4>`
* :doc:`chap5 — Clustering <../part2/chap5>`
* :doc:`chap6 — Frequent Itemset Mining (+ règles d'association) <../part2/chap6>`

.. note::
   Les algorithmes avancés (Partie 3) et le *sequential pattern mining* approfondi
   ne font **pas** partie de ce parcours.

Travaux & évaluation
====================

Tous les travaux notés peuvent se faire en groupe selon les consignes de l'enseignant.

* :doc:`TP1 — Analyse descriptive <../part4/index>` (un dataset assigné par étudiant)
* :doc:`TP2 — Implémentation d'Apriori <../part4/index>`
* :doc:`TP3 — Implémentation de K-means <../part4/index>`
* **TD médian** — un ancien examen (format proche de l'examen)
* **Examen**

.. note::
   Les **poids** de chaque élément sont communiqués par l'enseignant en début de
   semestre.
